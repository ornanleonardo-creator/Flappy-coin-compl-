# Flappy Coin

Un jeu inspiré de Flappy Bird, avec un système de monétisation basé sur AdMob.

## 📋 Fonctionnalités

- **Gameplay classique Flappy Bird**
- **Monétisation AdMob**
  - Bannières
  - Publicités interstitielles
  - Publicités récompensées
- **Thèmes et personnalisations**

---

## 📦 Configuration

1. Créer un compte AdMob: [AdMob Console](https://admob.google.com)
2. Ajouter les fichiers audio (wing.ogg, hit.ogg...).
